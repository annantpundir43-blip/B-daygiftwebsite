# 🎉 Birthday Gift Website

This is a fun birthday website I created using HTML and CSS. Instead of giving a normal birthday card, I wanted to make something interactive and memorable.

The website contains a series of gift boxes that reveal funny and entertaining GIFs when you hover over them. Each gift represents a different message or reaction, making the experience more personal and enjoyable.

## ✨ Features

* Interactive gift reveal effect
* Fun GIF animations
* Personalized birthday messages
* Clean and simple design
* Built entirely with HTML and CSS

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts

## 📚 What I Learned

While building this project, I practiced:

* CSS hover effects
* Background images
* Flexbox layouts
* Styling with custom fonts
* Creating interactive web pages without JavaScript

## 🚀 How to Run

1. Download or clone the repository.
2. Make sure all images and GIF files are in the project folder.
3. Open `index.html` in your browser.

That's it!

## 🎯 Why I Built This

I built this project to improve my front-end development skills and experiment with creative UI ideas. It was a fun way to learn CSS while creating something that feels more personal than a standard webpage.
